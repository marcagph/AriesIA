por Áudio e IA

Plataforma que registra atividades de trabalho via áudio no WhatsApp, interpreta automaticamente com IA e gera dashboards por equipe para gestores.

🟢 WhatsApp 🤖 IA (Claude / Whisper) 📊 Dashboard em tempo real 23 colaboradores
🎯
O Problema Atual
Por que a ferramenta atual não funciona bem
😤
Registro inconsistente
Cada pessoa registra de um jeito: alguns em tempo real, outros por semana, outros nunca. Dado pouco confiável.
🖥️
Formulário chato
Abrir o sistema, preencher 6 campos, clicar em Criar. Para 23 pessoas, várias vezes ao dia, é inviável.
🔒
Sem visibilidade
Colaboradores não veem seus próprios dados. Gestores veem tudo mas sem cruzamento entre equipes.
🔗
Desconectado do IAR
Os dados de tempo não se integram com nenhum outro sistema interno da empresa.
💡
A Solução
Simples como mandar um áudio pra um amigo
🎙️
Registro por Áudio
A pessoa manda um áudio no WhatsApp: "Gastei 2h em reunião com a empresa X e 30min respondendo e-mails." Pronto.
🤖
IA Interpreta e Categoriza
Whisper transcreve o áudio. Claude extrai os dados: atividade, tempo, empresa, categoria. Tudo automático.
📊
Dashboard para Gestores
Painel web com horas por pessoa, por equipe, por tipo de atividade e por cliente — em tempo real.
✅
Confirmação Instantânea
Após o registro, o bot responde: "✅ Registrado! Hoje você tem 5h30 registradas." A pessoa sabe que foi salvo.
📈

📈

Fluxo do Colaborador
Do áudio ao banco de dados em segundos
1
Colaborador manda um áudio no WhatsApp
Sem abrir nenhum sistema. Pode ser durante o dia ou no final. Texto também funciona.
🎙️ "Gastei 30 minutos mandando e-mail pra empresa X, 3 horas montando uma campanha de marketing e 2 horas fazendo configurações no novo site."
2
Whisper transcreve o áudio
A API do Whisper (OpenAI) converte o áudio em texto com alta precisão, mesmo com sotaque ou palavras do setor de RH.
3
Claude interpreta e estrutura os dados
A IA extrai automaticamente: atividade, tempo gasto, empresa/cliente e categoria. Retorna um JSON estruturado.
📋 Extraído: [E-mail empresa X → 0:30 → Comunicação] [Campanha marketing → 3:00 → Marketing] [Config. site → 2:00 → TI/Projetos]
4
Dados salvos no banco automaticamente
Cada atividade é registrada com: colaborador, equipe, data, hora, duração, categoria e cliente.
5
Confirmação enviada de volta
O colaborador recebe uma mensagem confirmando o registro, com o resumo do dia.
✅ Registrado! 3 atividades salvas. Hoje você tem 5h30 registradas. Bom trabalho!

Arquitetura Técnica
Camadas da solução e ferramentas recomendadas
Camada 1 — Entrada
WhatsApp
Z-API ou Evolution API
Recebe áudios e textos dos colaboradores
Camada 2 — Transcrição
Whisper API
OpenAI Whisper
Converte áudio em texto com alta precisão
Camada 3 — Interpretação
Claude AI
Anthropic Claude Sonnet
Extrai atividades, tempos e categorias do texto
Camada 4 — Orquestração
n8n ou Make
Plataforma de automação
Conecta todas as peças sem código pesado
Camada 5 — Banco de Dados
Supabase
PostgreSQL gerenciado
Grátis até 500 MB. Escala facilmente.
Camada 6 — Dashboard
Next.js + Vercel
App web hospedado
Grátis para projetos pequenos
Camada 7 — Integração
IAR / Senior / Rubi
Via webhook ou exportação
Conecta com sistemas existentes do Grupo
Camada 8 — Autenticação
Login por número WhatsApp
Cada colaborador é identificado pelo número — sem senha pra memorizar
⚡
Vantagem da arquitetura: Cada peça pode ser trocada sem reconstruir o resto. Começou com Google Sheets? Migra pro Supabase depois. Começou com n8n? Troca pro Make. Flexível e sem vendor lock-in.

Seu Passo a Passo
O que fazer primeiro — do zero ao sistema funcionando em 3 semanas
📋 Semana 1 — Criar as Contas
Use seu e-mail do Google. Plano gratuito já é suficiente pra começar.
Acesse z-api.io → crie instância → conecte via QR Code. 7 dias grátis para testar.
Acesse platform.openai.com → API Keys → Create new secret key. Guarde o código gerado.
Acesse console.anthropic.com → API Keys → Create Key. Guarde o código.
Crie duas abas: "Registros" (com colunas: Data, Colaborador, Equipe, Atividade, Tempo em minutos, Categoria, Cliente) e "Colaboradores" (Nome, Telefone, Equipe, Gestor).
Inclua o número de telefone com DDI (ex: 5531999999999), nome, equipe e gestor responsável.
⚙️ Semana 1–2 — Montar o Fluxo no Make
Nome sugerido: "Registro de Atividades - WhatsApp". Clique em "+ Criar cenário".
Adicione módulo "Webhooks → Custom webhook" → copie a URL gerada → cole no campo Webhook da Z-API.
Adicione "HTTP → Make a request" para baixar o arquivo de áudio usando a URL que chega no webhook.
Adicione "OpenAI → Create Transcription" → insira sua chave de API → modelo: whisper-1 → selecione o arquivo de áudio do bloco anterior.
Adicione "HTTP → Make a request" para chamar a API do Claude (api.anthropic.com/v1/messages). Cole o prompt da aba de Arquitetura e passe o texto transcrito como variável.
Adicione "Google Sheets → Add a Row" → conecte sua conta Google → selecione a planilha → mapeie os campos retornados pelo Claude para as colunas certas.
Adicione "HTTP → Make a request" para chamar a Z-API e enviar mensagem de volta: "✅ Registrado! X atividades salvas. Total hoje: Xh."
Mande um áudio de teste pelo WhatsApp e verifique se: o texto foi transcrito corretamente, os dados foram salvos na planilha e a confirmação chegou de volta.
📊 Semana 2–3 — Dashboard e Lançamento
Acesse lookerstudio.google.com → "+ Criar" → "Relatório" → conecte sua planilha como fonte de dados.
Barras de horas por colaborador, pizza de categorias, linha de evolução semanal, tabela de clientes e scorecard de total mensal.
Insira controles de filtro para: intervalo de datas, equipe, colaborador e categoria. Gestores vão usar esses filtros no dia a dia.
Duplique o relatório → adicione filtro fixo por equipe → compartilhe o link com cada gestor pelo e-mail.
Teste com você + 2 colegas. Verifique se a IA entende bem as atividades do Grupo Prestarh e ajuste as categorias se necessário.
Envie uma mensagem no grupo do WhatsApp explicando: "Agora é só mandar um áudio ou texto descrevendo o que você fez. A IA registra automaticamente!"
